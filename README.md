# ciunit.github.io

Source repository for **[CIunit.org](https://ciunit.org)**, the public-facing
website of the **Conceptual Investigations Unit** вЂ” a research consortium,
closely affiliated with the Stanford University Sustainable Solutions Lab,
that develops conceptual frameworks, quantitative analyses, and
interdisciplinary investigations at the intersection of climate, energy,
technology, and society.

The site itself is **static HTML/CSS**, served by GitHub Pages. Some of its
content is **generated by Python tooling** in this repository and committed
into the published `docs/` directory.

## How it works

GitHub Pages is configured to serve the site from the **`/docs` folder on the
`main` branch**. There is no server-side build step: whatever HTML/CSS/assets
are committed under `docs/` is exactly what gets published. The Python
generator runs **locally**, writes its output into `docs/`, and that output is
committed to git.

### Source vs. output

| Path                              | Role          | Who edits it     |
| --------------------------------- | ------------- | ---------------- |
| `content/`, `src/`                | **source**    | maintainers      |
| `docs/`                           | **build output** | the generator |

Hand-edited files and generated files both live in git, but they have
different owners. Where possible, the generator owns whole files or folders in
`docs/` so generated and hand-written content never need to be merged by hand.

## Building locally

> The Python tooling is being added incrementally; commands below reflect the
> intended workflow.

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .            # once pyproject.toml exists

python -m ciunit_gen.build  # regenerate content into docs/
```

Preview the static site without any tooling:

```bash
python -m http.server -d docs 8000
# open http://localhost:8000
```

## Deploying

Commit the regenerated `docs/` and push to `main`. GitHub Pages publishes the
new content automatically.

- **Custom domain:** `ciunit.org`, set via `docs/CNAME`.
- **Pages source:** `main` branch, `/docs` folder
  (Repo в†’ Settings в†’ Pages).
